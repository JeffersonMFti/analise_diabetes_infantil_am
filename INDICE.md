# 📂 ÍNDICE DO PROJETO - Análise Diabetes Infantil Amazonas

## 🎯 Acesso Rápido

### 🚀 Para Executar o Projeto:
```bash
python scripts\executar_simples.py
```

### 📧 Para Enviar Resultados:
- **Email completo**: `entrega\RESUMO_EXECUTIVO_EMAIL.md`
